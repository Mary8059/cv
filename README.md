# CV page (React + Vite)

Этот проект сделан на **React** и **Vite**.

## Что такое React простыми словами

React — это библиотека для интерфейсов.

- Вы описываете страницу как набор блоков (компонентов): `Header`, `ExperienceCard`, `Skills`.
- Данные храните в массивах/объектах и выводите через `map`, вместо копирования разметки вручную.
- Если меняете данные, интерфейс автоматически обновляется.

В этом проекте вся страница находится в `App.tsx`, а точка входа — `main.tsx`.

## Почему код из Figma мог подсвечиваться красным

Обычно причины такие:

1. Не были установлены зависимости (`npm install`).
2. В проекте не было корректной точки входа (`main.tsx`).
3. Открыта не корневая папка проекта в VS Code.
4. Плагин TypeScript/ESLint ругался на лишние или неиспользуемые элементы.

## Как запустить локально

```bash
npm install
npm run dev
```

После запуска откройте адрес из терминала (обычно `http://localhost:5173`).

## Как опубликовать на GitHub Pages

1. Создайте репозиторий и запушьте код.
2. Добавьте в `package.json` скрипты:

```json
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "deploy": "npm run build && npx gh-pages -d dist"
}
```

3. Установите пакет:

```bash
npm i -D gh-pages
```

4. Если сайт в репозитории `my-cv`, добавьте в `vite.config.ts`:

```ts
export default defineConfig({
  base: '/my-cv/',
  // ...
})
```

5. Опубликуйте:

```bash
npm run deploy
```

Если нужно, можно оставить только один файл `App.tsx` и редактировать его как обычный шаблон: менять текст, ссылки и массивы с опытом.
